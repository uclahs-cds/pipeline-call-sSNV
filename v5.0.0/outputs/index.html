How to run the pipeline

  1. Using the stable release stored under /hot/software/pipeline/pipeline-call-sSNV/Nextflow/release/ or the development version by cloning the GitHub repository to your machine.
  2. Fill in the params section of the config file and input YAML
  3. Run the pipeline using the Nextflow submission script with the command below:
python path/to/submit_nextflow_pipeline.py \
    --nextflow_script path/to/call-sSNV.nf \
    --nextflow_config path/to/nextflow\
    --nextflow_yaml path/to/input.yaml \
    --pipeline_run_name <sample_id> \
    --partition_type F72 \
    --email jdoe@mednet.ucla.edu

Notes:

The reference .fa file in config file should be the same with the reference genome that genereates the input bam files.
